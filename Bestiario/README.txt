BESTIÁRIO FORMATADO PARA FIRECAST

Como usar:
1. Abra o arquivo index.html no navegador.
2. Clique na criatura desejada.
3. Selecione o conteúdo visual da página renderizada, não o código-fonte.
4. Copie com Ctrl+C.
5. Cole no campo de texto rico do Firecast.

Observação:
Arquivos .txt são texto puro e sempre entram sem formatação. Estes .html existem para aproveitar a colagem rica do editor.
Se o Firecast bloquear estilos do navegador, a alternativa será montar um exportador no formato interno .bib/XML do Firecast.
