# Wiki Coliseu

Site estático de consulta do **Coliseu dos Campeões** para mestre e jogadores.

## Conteúdo publicado

- Bestiário completo em `Bestiario/`
- Mercador em `mercador.html`
- Itens iniciais em `itens-iniciais.html`

## Publicação

O deploy é feito pelo GitHub Pages via GitHub Actions usando `.github/workflows/pages.yml`.
